Cazalla

What is Cazalla?

Cazalla is a tool for generating static websites. It is coded in PHP using Symfony2 components, so it may be specially friendly to Symfony and Twig users.

Installation

To start using Cazalla, first clone its repository: 
 git clone git://github.com/nacmartin/Cazalla.git
Then update submodules to obtain the dependences 

cd Cazalla
git submodule init
git submodule update
Now you can start creating your site.

Creating a project

To create a project, run the following command: 

app/console cazalla:create --skeleton=tutorial
You can ommit the --skeleton option if you want to create an empty project
Now, in the project/ you will see a fresh new project created for you. If you want to compile it, run: 

php tutorial.php
And Cazalla will generate the statically generated content in project/output.
